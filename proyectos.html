Proyectos :

En este proyecto desarrollé una calculadora básica usando HTML, CSS y JavaScript. Permite realizar operaciones como suma, resta, multiplicación y división entre dos números ingresados mediante botones. El resultado se muestra en pantalla al pulsar “intro”, y se puede reiniciar con un botón de “reset”.

Con este proyecto practiqué la manipulación del DOM, eventos con addEventListener, control de flujo y el uso de funciones en JavaScript. Es una aplicación sencilla pero útil para entender la lógica detrás de las interacciones en la web.

En este ejercicio he creado un contador de clics con JavaScript que permite aumentar o disminuir un valor numérico al pulsar los botones. Además, he añadido una función que cambia aleatoriamente el color de fondo del cuerpo de la página usando valores RGB generados con Math.random(). Esta parte me ha permitido practicar cómo generar colores aleatorios y aplicarlos dinámicamente al estilo de la página.
